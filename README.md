This project is a fully interactive Python implementation of Conway's Game of Life. 
It provides a versatile experience where you can either define your own field dimensions and manually draw initial cell patterns or choose to load famous structures like the Glider, Gosper Glider Gun, and Pulsar. 
The application features a user-friendly interface that allows for real-time interaction, 
enabling you to toggle the state of individual cells with a mouse click and use the spacebar to pause or start the simulation as it evolves. 
The game ensures that moving objects wrap around the grid edges.

Run the main script: `life.py`
